# 🏦 Banking Management System

A full-stack Banking Management System built using Java Spring Boot, MySQL, HTML, CSS, JavaScript, and Razorpay Test Payment Integration.

---

# 🚀 Features

- User Registration & Login
- Admin Dashboard
- User Dashboard
- Create Bank Account
- Deposit Money
- Withdraw Money
- Razorpay Test Payment Integration
- Transaction History
- Email Notifications
- Secure User Account Access
- Responsive UI
- Local Network Project Sharing

---

# 🛠 Technologies Used

## Backend
- Java
- Spring Boot
- Spring Data JPA
- MySQL
- Maven

## Frontend
- HTML
- CSS
- JavaScript

## Payment Gateway
- Razorpay Test API

# ⚙️ How To Run

## 1. Clone Repository

```bash
git clone https://github.com/your-username/banking-management-system.git
```

---

## 2. Open Project

Open project in:
- STS
- Eclipse
- IntelliJ IDEA

---

## 3. Configure Database

Create MySQL database.

Example:

```sql
CREATE DATABASE mybankingapp;
```

---

## 4. Configure application.properties

Add:
- MySQL username
- MySQL password
- Razorpay test keys
- Email credentials

---

## 5. Run Spring Boot Application

Run:

```text
MybankingappApplication.java
```

---

## 6. Open Application

Frontend:

```text
http://localhost:8080
```

OR

```text
http://YOUR_IP_ADDRESS:8080
```

for same Wi-Fi access.

---

# 🔐 Important Note

Do NOT upload:
- application.properties
- Razorpay Secret Key
- Database Passwords
- Gmail App Password

to public GitHub repositories.
